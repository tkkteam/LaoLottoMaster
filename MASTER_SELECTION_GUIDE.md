# 🎯 MASTER SELECTION (ชุดเด่น) - อัพเกรดใหม่

## 📊 หลักการคำนวณที่ใช้

### ✨ MASTER ENSEMBLE (สูตรรวมพลัง) - สูตรใหม่ที่แม่นยำที่สุด

ใช้หลักการ 5 ประการร่วมกัน:

#### 1. **Hot Numbers Analysis (30%)**
- วิเคราะห์เลขที่ออกบ่อยในช่วง 30 งวดล่าสุด
- แยกวิเคราะห์หลักสิบและหลักหน่วย
- ให้คะแนนตามความถี่ (เลขที่ออกบ่อยได้คะแนนสูง)

#### 2. **Pattern Matching (25%)**
- หารูปแบบที่ซ้ำในข้อมูลย้อนหลัง
- เมื่อหลักสิบ/หลัก_unit ตรงกับอดีต ดูว่างวดหน้าออกอะไร
- ให้คะแนนกับเลขที่เคยเกิด 패턴 เดียวกัน

#### 3. **Trend Following (20%)**
- ดูแนวโน้ม 5 งวดล่าสุด
- งวดที่ใหม่กว่ามีน้ำหนักมากกว่า
- ตามทิศทางปัจจุบัน (momentum)

#### 4. **Gap Analysis (15%)**
- วิเคราะห์ช่องว่างการออก
- เลขที่หายไปนาน มีโอกาสออกสูง (due theory)
- นับจำนวนงวดตั้งแต่ lầnล่าสุดที่ออก

#### 5. **Mirror Logic (10%)**
- ใช้หลักการเลขกระจก (0↔5, 1↔6, 2↔7, 3↔8, 4↔9)
- ให้คะแนนกับเลขกระจกของงวดก่อนหน้า

### 📈 การให้คะแนน
- แต่ละหลักการให้คะแนน 0-100 กับเลข 0-9
- รวมคะแนนทั้งหมด (weighted sum)
- เลือกเลขที่ได้คะแนนสูงสุดเป็นหลักสิบและหลักหน่วย

## 🔍 ระบบกรองสูตรอัตโนมัติ (Consecutive Hit Filter)

### หลักการ
- **ทดสอบย้อนหลัง 30 งวด**
- **นับจำนวนครั้งทายถูกติดต่อกันสูงสุด (Max Consecutive Hits)**
- **เกณฑ์ผ่าน: ต้องทายถูกติดต่อกันอย่างน้อย 6 งวด**

### การทำงาน
```
❌ สูตรที่ไม่าผ่านเกณฑ์ (< 6 งวดติด) → ถูกลบออก
✅ สูตรที่ผ่านเกณฑ์ (≥ 6 งวดติด) → แสดงใน Leaderboard
🏆 สูตรที่ดีที่สุด → ถูกเลือกเป็น Active Master
```

### ตัวอย่าง Output ใน Console

## 🎨 การแสดงผลใน UI

### Leaderboard แสดง 3 สถานะ:

#### 🟢 **Active Master** (สีเขียว)
- สูตรที่ดีที่สุดในขณะนี้
- ถูกเลือกอัตโนมัติตามความแม่นยำ + consecutive hits
- พื้นหลังสีไล่โทนเขียว

#### 🔵 **Qualified** (สีฟ้า)
- ผ่านเกณฑ์ 6 งวดติด
- แต่ยังไม่ใช่ดีที่สุด
- มี badge "✓ 6+ Consecutive"

#### 🔴 **Failed Test** (สีแดง)
- ไม่ผ่านเกณฑ์ (< 6 งวดติด)
- มีความน่าเชื่อถือต่ำ
- มี badge "✗ Failed Test"

### ข้อมูลที่แสดง
- **Max Consecutive**: จำนวนงวดที่ทายถูกติดต่อกันสูงสุด
- **Direct Accuracy**: % ทายถูกตรงๆ
- **Running Accuracy**: % ทายถูกทั้งหลักสิบหรือหลักหน่วย
- **Confidence**: คะแนนรวมทั้งหมด

### สำหรับผู้ใช้
1. เปิดเว็บแอปพลิเคชัน
2. กด **"SYNC SYSTEM"** เพื่อดึงข้อมูลล่าสุด
3. ระบบจะ:
   - ทดสอบสูตรทั้งหมดอัตโนมัติ
   - กรองสูตรที่ไม่ผ่านเกณฑ์ออก
   - เลือกสูตรที่ดีที่สุดให้
4. ดูผลลัพธ์ใน:
   - **AI MASTER**: ตัวเลขที่สูตรแนะนำ
   - **MASTER SELECTION (ชุดเด่น)**: เลขเด่น 8 ตัว
   - **Leaderboard**: อันดับและสถานะทุกสูตร

### สำหรับ Developer
```typescript
// เรียกใช้ฟังก์ชันกรองสูตร
import { filterPatternsByConsecutive, fetchLottoData } from './services/lottoService';

const data = await fetchLottoData();
const qualified = filterPatternsByConsecutive(data, 6, 30); // 6 งวดติด, 30 งวดทดสอบ

// ใช้สูตรที่ดีที่สุด
const bestPattern = qualified[0].pattern;
const prediction = bestPattern.calc(
  parseInt(data[1].r2, 10),  // งวดก่อนหน้า
  parseInt(data[0].r2, 10),  // งวดล่าสุด  
  data[0].r4,                // 4 หลัก
  data                       // ข้อมูลทั้งหมด
);
```

## 🔧 Technical Details

### BacktestResult Interface
```typescript
interface BacktestResult {
  totalRounds: number;           // จำนวนรอบที่ทดสอบ
  directHits: number;            // ทายถูกตรงๆ
  runningHits: number;           // ทายถูกหลักสิบหรือหลักหน่วย
  directAccuracy: number;        // % ตรง
  runningAccuracy: number;       // % วิ่ง
  maxConsecutiveHits: number;    // 🆕 ถูกติดต่อกันสูงสุด
  hits: Array<{...}>;            // รายละเอียดแต่ละรอบ
}
```

### ฟังก์ชันสำคัญ
- `filterPatternsByConsecutive()` - กรองสูตรตาม consecutive hits
- `backtestPattern()` - ทดสอบสูตรย้อนหลัง
- `findBestPattern()` - หาสูตรที่ดีที่สุด
- `MASTER ENSEMBLE calc()` - คำนวณแบบผสมผสาน
