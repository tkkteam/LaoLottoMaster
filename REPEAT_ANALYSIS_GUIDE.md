# 🔄 REPEAT ANALYSIS - วิเคราะห์โอกาสออกซ้ำ

## 📊 คำถามของคุณ
> "สูตรคำนวณได้เลข **10** และงวด 01/04/2569 ก็ออก **10** เช่นกัน แล้วโอกาสที่เลข 10 จะออกซ้ำอีกมีมากน้อยแค่ไหน?"

---

## 🎯 คำตอบ: ขึ้นอยู่กับสถิติย้อนหลัง!

ระบบ **Repeat Analysis** จะวิเคราะห์ข้อมูลย้อนหลัง 100 งวด เพื่อตอบคำถามนี้ให้ชัดเจน:

---

## 📈 หลักการทำงาน

### **สิ่งที่วิเคราะห์:**

1. **Total Occurrences** - เลข 10 ออกทั้งหมดกี่ครั้งใน 100 งวด?
2. **Repeat After One** - ออกซ้ำในงวดถัดไปกี่ครั้ง?
3. **Repeat After Two** - ออกซ้ำใน 2 งวดถัดไปกี่ครั้ง?
4. **Repeat After Three** - ออกซ้ำใน 3 งวดถัดไปกี่ครั้ง?
5. **Repeat Percentage** - % การออกซ้ำในงวดถัดไป
6. **Average Gap** - โดยเฉลี่ยออกทุกกี่งวด?
7. **Last Seen** - ครั้งล่าสุดที่ออกคืองวดไหน?

---

## 🎨 การแสดงผลใน UI

### **การ์ดวิเคราะห์สี 3 ระดับ:**

#### 🔴 **HIGH (โอกาสออกซ้ำสูง)**
- **Repeat Rate > 20%**
- ขอบสีแดง
- ⚠️ คำเตือน: "เลข XX มีแนวโน้มออกซ้ำสูง (X%)"
- **คำแนะนำ**: ระวัง! อาจออกซ้ำ แต่ไม่ควรพึ่งมากเกินไป

#### 🟡 **MEDIUM (โอกาสออกซ้ำปานกลาง)**
- **Repeat Rate 10-20%**
- ขอบสีเหลือง
- 🔶 คำเตือน: "เลข XX มีโอกาสออกซ้ำปานกลาง (X%)"
- **คำแนะนำ**: พิจารณาร่วมกับสูตรอื่น

#### 🟢 **LOW (โอกาสออกซ้ำต่ำ)**
- **Repeat Rate < 10%**
- ขอบสีเขียว
- ✅ คำแนะนำ: "เลข XX มีโอกาสออกซ้ำต่ำ (X%) ควรเลี่ยง"
- **คำแนะนำ**: ไม่ควรใช้เลขนี้ถ้าไม่อยากได้เลขซ้ำ

## 📊 ตัวอย่างการคำนวณ

### **กรณีศึกษา: เลข 10**

```
ข้อมูลย้อนหลัง 100 งวด:
- เลข 10 ออกทั้งหมด: 8 ครั้ง
- ออกซ้ำงวดถัดไป: 1 ครั้ง
- Repeat Rate: 1/7 = 14.3% (MEDIUM)
- Average Gap: 12.5 งวด
- Last Seen: 01/04/2569

ผลลัพธ์:
🔶 เลข 10 มีโอกาสออกซ้ำปานกลาง (14.3%)
→ พิจารณาร่วมกับสูตรอื่นได้ แต่ไม่ควรพึ่งอย่างเดียว
```

---

## 🧮 สูตรคำนวณ

### **Repeat Percentage**
```
Repeat % = (Repeat After One / (Total Occurrences - 1)) × 100
```

**ตัวอย่าง:**
- Total Occurrences: 8 ครั้ง
- Repeat After One: 1 ครั้ง
- Repeat % = (1 / 7) × 100 = **14.3%**

### **Average Gap**
```
Avg Gap = ผลรวมช่องว่างระหว่างการออก / จำนวนครั้ง - 1
```

**ตัวอย่าง:**
- ออกครั้งที่: งวดที่ 5, 18, 30, 42, 55, 68, 80, 95
- Gaps: 13, 12, 12, 13, 13, 12, 15
- Avg Gap = (13+12+12+13+13+12+15) / 7 = **12.86 งวด**

## 📋 การตีความผลลัพธ์

### **ถ้า Repeat Rate สูง (> 20%)** 🔴
- ✅ **เลขนี้มีแนวโน้้มออกซ้ำบ่อย**
- ⚠️ แต่ไม่ควรพึ่งมากเกินไป เพราะยังมีความสุ่ม
- 💡 ใช้ร่วมกับ Hot Numbers จะดีกว่า

### **ถ้า Repeat Rate ปานกลาง (10-20%)** 🟡
- ✅ **มีโอกาสออกซ้ำบ้าง แต่ไม่บ่อย**
- 💡 พิจารณาร่วมกับสูตรอื่น
- 💡 ใช้เป็นเลขสำรองได้

### **ถ้า Repeat Rate ต่ำ (< 10%)** 🟢
- ✅ **มีโอกาสออกซ้ำต่ำมาก**
- 💡 ควรเลี่ยงถ้าไม่ต้องการเลขซ้ำ
- 💡 อาจพิจารณาเลขใกล้เคียงแทน

## 🔧 Technical Details

### **ฟังก์ชัน**
```typescript
analyzeRepeatProbability(
  results: LottoResult[],     // ข้อมูลหวยทั้งหมด
  targetNumber: string,       // เลขที่ต้องการวิเคราะห์ (เช่น "10")
  lookbackRounds: number = 100 // ย้อนหลังกี่งวด
): RepeatAnalysis
```

### **Interface**
```typescript
interface RepeatAnalysis {
  totalOccurrences: number;      // จำนวนครั้งที่ออกทั้งหมด
  repeatAfterOne: number;        // ออกซ้ำงวดถัดไป
  repeatAfterTwo: number;        // ออกซ้ำ 2 งวดถัดไป
  repeatAfterThree: number;      // ออกซ้ำ 3 งวดถัดไป
  repeatPercentage: number;      // % ออกซ้ำ
  averageGap: number;            // ช่องว่างเฉลี่ย
  lastSeenDate: string;          // วันที่ออกครั้งล่าสุด
  confidenceLevel: 'HIGH' | 'MEDIUM' | 'LOW';
  recommendation: string;
}
```

### **การใช้งาน**
```typescript
import { analyzeRepeatProbability } from './services/lottoService';

const lastR2 = allData[0].r2; // "10"
const analysis = analyzeRepeatProbability(allData, lastR2, 100);

console.log(analysis.repeatPercentage);    // 14.3
console.log(analysis.confidenceLevel);     // 'MEDIUM'
console.log(analysis.recommendation);      // 🔶 เลข 10 มีโอกาส...
```

## 📝 สรุป

### ✅ **สิ่งที่ระบบทำ:**
1. ตรวจสอบย้อนหลัง 100 งวด
2. นับจำนวนครั้งที่เลขออกและออกซ้ำ
3. คำนวณ % โอกาสออกซ้ำ
4. ให้คำแนะนำอัตโนมัติ

### 🎯 **สิ่งที่คุณทำได้:**
1. ดู Repeat Rate % เพื่อประเมินโอกาส
2. ใช้ร่วมกับสูตรอื่น (MASTER SELECTION, AI MASTER)
3. ตัดสินใจด้วยข้อมูล ไม่ใช่แค่ความรู้สึก

### ⚠️ **ข้อควรระวัง:**
- หวยยังคงมีความสุ่ม ไม่มีสูตรไหน 100%
- Repeat Rate เป็นสถิติย้อนหลัง ไม่ได้การันตีอนาคต
- ใช้เป็น **ข้อมูลประกอบ** เท่านั้น
